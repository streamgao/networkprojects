# Examining forms

## Tools

This uses the following tools and libraries:

  * express
  * express.static
  * [body-parser](https://github.com/expressjs/body-parser)
  * util

## Install

  * `git clone https://github.com/DWDatITP/sample-forms.git`
  * `cd sample-forms`
  * `npm install`
  * `nodemon server.js`
  * visit <http://localhost:5000>
